# Trabajo Final - Díaz Marcos

Trabajo final de integración de la material Construcción de Interfaces de Usuario de la UNQ

## Instalación

> [!IMPORTANT]
> Debe tener instalado previamente Node.js

1. Ir a la sección de _Tags_ de Github
2. Buscar la release más reciente y descargar el comprimido
3. Extraer el comprimido y luego entrar en la carpeta generada.
4. abrir una terminal CMD y escribir el siguiente comando

```
npm install
```

Y para levantar la aplicación en la misma terminal escribís este comando:

```
npm run dev
```

Luego en tu navegador, ingresá en: `localhost:5173`.
